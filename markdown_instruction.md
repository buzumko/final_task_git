## Списки

Чтобы добавить ненумерованный список, необходимо пункты выделять звездочкой (*). Например, вот так:
* элемент 1; 
* элемент 2; 
* элемент 3.

Чтобы добавить нумерованный список, необходимо пункты просто пронумеровать. Например, вот так:
1. Элемент 1. 
2. Элемент 2.
