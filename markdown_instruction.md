# Инструкция для работы с MD

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его зведочками (*). Например, *вот так*

Чтобы выделить текст жирным необходимо обрамить его двумя зведочками (**). Например, **вот так**
